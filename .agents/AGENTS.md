# Agent Rules

## Release Process per GestionAle
Quando vengono aggiunte nuove funzionalità, pagine o automazioni a GestionAle, DEVI sempre fare quanto segue prima di considerare concluso il task:
1. **Aggiornare la versione**: Cerca la stringa della versione corrente in `src/components/DashboardClient.js` (es. "v2.2.0") e incrementala (es. "v2.3.0").
2. **Aggiornare le Note di Rilascio**: Vai in `src/components/HelpModal.js` e aggiungi un elenco puntato dettagliato (usando emoji) che descriva in modo chiaro le nuove funzionalità appena rilasciate (es. "🚀 Note di Rilascio (vX.X.X)").
Queste operazioni sono essenziali per il cliente. Fallo sempre senza che ti venga chiesto!
